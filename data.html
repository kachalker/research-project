Data Model:

 All of the data for this project came from online sources. For the first dataset I created, I utilized data from Tony Scupham–Bilton's blog, The Queerstory Files. On his blog, Scupham–Bilton compiled a list of every LGBTQ+ Olympic athlete and included information about them such as their name, home country, sport, what Olympics they competed in, and the medals they won. On his lists, along with the actual Olympic athletes, he also included reserve and alternate athletes as well as any athletes who competed in any Olympic trial or qualifying event. For the purpose of this project, I only included the actual Olympic athletes and reserve and alternate athletes since athletes who competed in the trials were not affected at all by the host country. This data is helpful to my research project because it provided me with all of the data about the athletes that I need.

 The second dataset I created utilized data from The World Population Review and Equaldex. The data I used from The World Population Review included the name of every world country and the population of that country in 2020. I chose to include this data in my dataset because it will allow me to put the number of LGBTQ+ athletes into context based on the countryâ€™s total population. The data I utilized from Equaldex was every country's equality index score. The equality index score is a 0–100 rating given to each country that demonstrates the rights and protections afforded to the LGBTQ+ community. This score includes things such as the legality of homosexuality and homosexual marriage, legal protections given to the LGBTQ+ community against discrimination, and the recognition of gender changes or non-binary genders. I chose to include the data from Equaldex into my dataset because it provides me a way to quanitifably say how LGBTQ+ friendly a country is.

Data Plan:

 To create my first dataset, I had to transform all of Scupham–Bilton's data into a CSV file since his was orginally a pdf file. To create this CSV file, I had to input all of the data piece by piece into Microsoft Excel. The only part of his data that I could not directly input was the Host Country because Scupham–Bilton only included the Host City with his data. However, to change this for my own dataset, I only had to look up on Google what country the city was located in.

 To create my second dataset, I first downloaded a CSV file from The World Population Review. Since The World Population Review Data also included data fields that I did not need for my project, I had to delete every data field except for Country and 2020 Population. Then I inputted the Equality Index Score from Equaldex as another data field on the same CSV file. Equaldex did not provide a comprehensize list of Equality Index Scores, so I had to input them country by country.
